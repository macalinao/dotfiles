# One Camera One Lens — darktable styles

15 scene-referred `.dtstyle` presets (v1.1) by Mark G. Adams of One Camera One
Lens. Vendored here because they are distributed via a Google Drive folder with
no stated license and cannot be pinned/fetched declaratively like the
t3mujinpack pack.

- Source page: https://onecameraonelens.com/2022/10/13/a-selection-of-darktable-styles/
- Original folder: https://drive.google.com/drive/folders/1WS4bSYuf8sz3avTebABojPjfI1YqeBLf
- Downloaded: 2026-06-01

These assume darktable's modern "scene-referred" workspace. They are staged to
`~/.config/darktable/styles-packs/onecameraonelens/` by
`nix/home/os-specific/darwin.nix`; import them once via Lighttable → styles →
import (darktable does not auto-load styles from disk).
